Hangman Game

Description:

This is a hangman game. The word my group decided to use was Strawberry. Everytime the user inserts the wrong letter the program will print out a body part of our little stick figure. As the program presents a body part the counter we have set up will count up till it reaches the number 6. This gives our user 6 tries till the game is over.
